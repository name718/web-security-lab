# 实验列表

## 注入攻击

### [XSS 跨站脚本](/labs/xss)
通过在网页中注入恶意脚本，窃取用户信息或执行恶意操作。

- 反射型 XSS
- 存储型 XSS
- Token 窃取

### [SQL 注入](/labs/sql-injection)
通过在输入中插入 SQL 代码，操纵数据库查询。

- 登录绕过
- UNION 注入
- 数据窃取

## 身份与会话

### [CSRF 跨站请求伪造](/labs/csrf)
利用用户已登录的身份，执行非预期操作。

- 模拟银行转账
- 钓鱼页面攻击
- Token 防护

### [JWT 安全漏洞](/labs/jwt)
JSON Web Token 的常见安全问题。

- 算法混淆 (alg: none)
- 弱密钥爆破
- 敏感信息泄露

### [钓鱼攻击](/labs/phishing)
通过伪造可信网站窃取用户凭证。

- 仿冒登录页
- 凭证窃取
- 识别技巧

## 权限控制

### [IDOR 对象级越权](/labs/idor)
通过修改对象 ID 访问他人数据。

- 查看他人订单
- 修改他人资料
- 删除他人文件

### [RBAC 权限模型缺陷](/labs/rbac)
权限设计和实现中的常见问题。

- 前端校验绕过
- 角色伪造
- 权限继承错误

## 可用性攻击

### [DDoS 拒绝服务](/labs/ddos)
通过大量请求耗尽服务器资源。

- HTTP Flood
- 限流防护
