🛡️ 安全设计原则

  • 后端校验 - 永远不要只依赖前端
  • 最小权限 - 只给必要的权限
  • 角色分离 - 管理员和普通用户严格区分
  • 审计日志 - 记录所有敏感操作
  • 定期审查 - 检查权限配置是否合理
