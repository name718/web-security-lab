什么是 SQL 注入？

SQL注入是一种代码注入攻击，攻击者通过在输入中插入恶意 SQL 代码，操纵数据库查询，从而：

绕过认证

无需密码直接登录管理员账户

数据泄露

窃取数据库中的敏感信息

数据篡改

修改、删除数据库记录

🛡️ 防御方案

  • 参数化查询 - 使用 Prepared Statements
  • ORM 框架 - 如 Sequelize、TypeORM
  • 输入验证 - 白名单过滤
  • 最小权限 - 数据库账户只给必要权限
  • 错误处理 - 不暴露数据库细节
