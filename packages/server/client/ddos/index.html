什么是 DDoS？

DDoS（分布式拒绝服务）攻击通过大量请求耗尽服务器资源，使正常用户无法访问服务。

HTTP Flood

发送大量 HTTP 请求耗尽服务器处理能力

Slowloris

慢速发送请求，长时间占用连接

SYN Flood

发送大量 TCP SYN 包（需系统级模拟）

📊 攻击类型对比

类型原理特点
HTTP Flood 大量 HTTP 请求 简单粗暴，易被识别
Slowloris 慢速发送，占用连接 低带宽，难检测
CC 攻击 请求耗资源的页面 针对应用层

🛡️ 防御方案

  • Rate Limiting - 限制单 IP 请求频率
  • CDN - 分散流量，隐藏源站
  • WAF - Web 应用防火墙
  • 黑洞路由 - 丢弃攻击流量
  • 验证码 - 区分人机
